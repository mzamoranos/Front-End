ClojureScript

ClojureScript es un compilador para Clojure, un Lips funcional, que se compila en JavaScript.

PROS

Tiene excelentes herramientas para el desarrollo web. Utiliza excelentes envoltorios alrededor de React.js que facilitan la creación de aplicaciones de una sola página.

Puede agregar Garden para hacer la escritura de CSS parte del mismo desarrollo, generando un proceso agradable y sin problemas.

CONTRAS

La sintaxis puede parecer críptica para las personas que no están acostumbradas a Lisp.

A Lisp a veces se le llama "sin sintaxis" y esto es desconcertante para los que esán inmersos en la sintaxis de tipo Algol (Java, Javascript, C, etc.)
